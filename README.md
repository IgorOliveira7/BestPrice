# BestPrice ⛽

Projeto desenvolvido durante o curso "Análise e Desenvolvimento de Sistemas" - do SENAC São Paulo.

O programa consiste em informar para o usuário qual combustível está mais em conta de acordo com o preço.

✔️ JavaScript<br>
✔️ HTML<br>
✔️ CSS<br>
✔️ Bootstrap
